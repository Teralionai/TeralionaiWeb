# Design Context — 兆獅智能 Teralionai

## Target Audience
企業決策者（CTO、IT 主管、投資部門主管）正在尋找 AI 轉型方案，尤其在金融程式交易領域。

## Use Cases
1. 了解兆獅智能的三大產品（AI 營業員、程式交易大師、Owl AI PM）
2. 評估是否適合自己的企業需求
3. 透過表單或 email 聯繫諮詢

## Brand Personality
科技感、專業權威、精確。不浮誇，不花俏。像一份高品質的技術白皮書 — 乾淨、有結構、值得信任。

## Aesthetic Direction: "Editorial Tech"
- **Typography**: Space Grotesk (display) + DM Sans (body)
- **Color**: 深色底 (#0b1219) + 精準的 teal 強調色 (#4ca8be)，不用 glow/text-shadow
- **Layout**: 不對稱、編輯排版風格，左對齊為主
- **Details**: 細線分隔、numbered 序號、clean borders，不用 glassmorphism/glow
- **Motion**: 克制的 scroll reveal + CSS grid 展開動畫
